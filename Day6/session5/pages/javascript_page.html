JavaScript Basics

Variables Data Types Functions Loops Conditions Events

Variables

Variables store data values in JavaScript. Use let, const, or var. let allows block-scoped variables, const declares constants, and var is function-scoped.


    let name = "Vaishnavi";
    const age = 24;
    var city = "Mumbai";

Data Types

JavaScript supports primitive types: string, number, boolean, null, undefined. Objects and arrays store multiple values.


    let name = "Vaishnavi";            // string
    let age = 24;                      // number
    let isStudent = true;              // boolean
    let person = {
                   name:"Vaishnavi", 
                   age:24
              };                         // object
    let numbers = [1,2,3];               // array

Functions

Functions are reusable blocks of code. There are three types: Function declarations, function expressions, and arrow functions.

// Declaration

function greet(name) {
  return "Hello " + name;
}

// Expression
const greet2 = function(name) {
  return "Hi " + name;
}

// Arrow
const greet3 = name => "Hey " + name;

Loops

Loops execute code repeatedly. Common loops include for, while, and do-while.

  

    for(let i=0; i<5; i++){
                            console.log(i);
                       }

             let i = 0;
                  while(i<5){
                  console.log(i);
                   i++;
            }

Conditions

Conditions control program flow. Use if, else if, else, or switch statements.



    let age = 18;
    if(age >= 18){
       console.log("Adult");
   } else 
   {
      console.log("Minor");
  }

switch(age){
   case 18: console.log("Just Adult"); 
   break;
   default: console.log("Other age");
   }

Events

Events let JavaScript respond to user actions like clicks, key presses, or mouse movements.


    button.addEventListener("click", function()
    {
         alert("Button clicked!");
  });

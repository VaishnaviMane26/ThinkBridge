CSS Syntax

CSS rules consist of a selector and a declaration block. The selector chooses the element to style, and the declaration block contains property-value pairs.


selector {
  property: value;
}

Selectors

Selectors target HTML elements. Examples include element selectors, class selectors, ID selectors, descendant selectors, and pseudo-classes for hover or focus effects.



p { 
    color: red;
 }       /* element */

#myId { 
     color: blue;
   }  /* ID */

.myClass {
     color: green;
 } /* class */

a:hover { 
    text-decoration: underline;
 } /* pseudo-class */

Colors & Backgrounds

CSS allows setting text and background colors using names, hex, RGB, or RGBA. Background images can also be applied and controlled with size, repeat, and position.


p {
  color: #333;                     /* text color */
  background-color: #f9f9f9;       /* background color */
  background-image: url('bg.jpg'); /* background image */
  background-size: cover;
}

Fonts & Text

Font family, size, weight, style, alignment, decoration, spacing, and line-height can all be controlled using CSS to make text readable and stylish.


p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  font-style: italic;
  text-align: center;
  text-decoration: underline;
  line-height: 1.5;
}

Box Model

Every element has content, padding, border, and margin. Box-sizing determines whether padding and border are included in the element's total width and height.


div {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
  box-sizing: border-box;
}

Display & Position

Elements can be displayed as block, inline, or inline-block. Positioning includes static, relative, absolute, fixed, and sticky, controlling the element's placement on the page.


div { 
    display: block;
 }

span {
       display: inline;
    }

li { 
    display: inline-block;
 }

.relative { 
    position: relative;
    top: 10px; 
    left: 10px;
 }

.absolute { 
    position: absolute;
    top: 50px; 
    right: 20px; 
}

.fixed {
     position: fixed;
     top: 0; 
    }

.sticky { 
          position: sticky;
          top: 0; 
    }

Flexbox

Flexbox provides a way to layout elements in a row or column, aligning and distributing space efficiently. It supports justify-content, align-items, and gap properties.


.container {
  display: flex;
  flex-direction: row;
  justify-content: space-around;
  align-items: center;
  gap: 10px;
}

Grid

CSS Grid allows creating complex layouts with rows and columns. Grid-template-columns, grid-gap, and grid-template-areas define the structure for content placement.


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 10px;
}
.grid-item {
  padding: 20px;
  text-align: center;
}

Lists

CSS can style ordered and unordered lists. List-style-type, padding, margin, and custom icons allow lists to match the design.


ul { 
    list-style-type: disc; 
}

ol { 
    list-style-type: decimal; 
}
ul li { 
         margin: 5px 0;
     }

Tables

Tables can be styled with borders, padding, background colors, and text alignment to make them clean and readable.


table {
          border-collapse: collapse; 
          width: 80%; 
    }

th, td {
          border: 1px solid #ddd; 
          padding: 8px; 
          text-align: center; 
        }

th {
     background-color: #34495e; 
     color: white;
 }

Forms

Input fields, text areas, buttons, and labels can be styled with colors, padding, borders, and hover effects for better usability.


input, textarea {
  padding: 8px;
  border-radius: 5px;
  border: 1px solid #ccc;
}
button {
  padding: 10px;
  background-color: #27ae60;
  color: white;
  border-radius: 5px;
  cursor: pointer;
}
button:hover { background-color: #219150; }

Animations

Transitions and keyframe animations allow elements to move, fade, or change styles smoothly, adding interactivity and visual interest.


button {
  transition: background 0.3s, transform 0.3s;
}
button:hover { transform: scale(1.1); }

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

Shadows & Gradients

Box-shadow and text-shadow add depth. Gradients create smooth color transitions for backgrounds or text, enhancing visual appeal.


div {
     box-shadow: 2px 2px 10px rgba(0,0,0,0.3);
     }
p { 
    text-shadow: 2px 2px #aaa;
 }
div { 
    background: linear-gradient(to right, #e67e22, #f1c40f);
 }

Media Queries

Media queries allow responsive designs by applying different styles based on screen width, height, or device type.


body { font-size: 14px; }
@media (min-width: 768px) { 
     body { 
        font-size: 16px;
     } }
@media (min-width: 1024px) { 
                  body { font-size: 18px; }
                 }

CSS Variables

Variables store reusable values like colors, fonts, or spacing. They help maintain consistency and simplify updates across the CSS file.


:root {
  --primary-color: #3498db;
  --secondary-color: #e67e22;
}
div { background-color: var(--primary-color);
     color: white; }
